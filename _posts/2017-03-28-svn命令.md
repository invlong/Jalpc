>今天SVN上传代码，之前有用，但是没有记录，所以给忘了。。。今天再次使用，以作记录。  

1.从远程仓库下载代码
-
这里的代码其实是一个空目录。  
``` shell
svn checkout svn://139.224.120.89/root/pos/pos-view
```
2.讲代码添加到仓库
--
``` shell
svn add .
```
3.提交代码
--
```shell
svn commit .
```
